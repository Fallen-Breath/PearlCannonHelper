PearlCannonHelper
-------------

A tool to simulate pearl's trace and generate setting from specific coordinate for [360FTL Heavy](https://youtu.be/1_MLMXVpHbU)

Works on Windows

-----------------

一个用于模拟珍珠轨迹或者由指定坐标生成[重型矢量珍珠炮](https://www.bilibili.com/video/BV1NC4y1x7WW)配置的工具

在Windows上运行
